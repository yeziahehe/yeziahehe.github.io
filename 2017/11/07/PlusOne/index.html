Plus One

2017.11.07 叶帆 LeetCode OJ  热度 ℃

题目介绍

LeetCode 66. Plus One，题目的意思给定一个非负数组表示一个数字，从高位向低位排，加一之后输出结果。

解题思路

先把数组反向然后遍历，如果是 index == 0 就加一，记录下进位值 carry = sum / 10 和加一后的当前数字 reDigits[index] = sum % 10。
