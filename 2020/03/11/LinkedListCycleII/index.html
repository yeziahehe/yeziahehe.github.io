Linked List Cycle II

2020.03.11 叶帆 LeetCode OJ  热度 ℃

题目介绍

LeetCode 142. Linked List Cycle II。

复杂度

时间复杂度: O(n), 空间复杂度: O(1)

解题思路

先看下 Linked List Cycle 的题解，找到入环节点是个数学问题。
第一次相遇后，fast 指针移到 head，然后和 slow 指针一起一次移动一步，再次相遇时的节点就是入环节点。
