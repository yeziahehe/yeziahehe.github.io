Linked List Cycle

2020.03.11 叶帆 LeetCode OJ  热度 ℃

题目介绍

LeetCode 141. Linked List Cycle。

复杂度

时间复杂度: O(n), 空间复杂度: O(1)

解题思路

算法学习 - 链表 对这题有扩展。

这是一道经典的链表题目 - 环形链表，基本的解题思路是双指针技巧里面的快慢指针。

  • 如果没有环，快指针将停在链表的末尾。
  • 如果有环，快指针最终将与慢指针相遇。

这两个指针的适当速度应该是多少？

一个安全的选择是每次移动慢指针一步，而移动快指针两步。每一次迭代，快速指针将额外移动一步。如果环的长度为 M，经过 M 次迭代后，快指针肯定会多绕环一周，并赶上慢指针。
