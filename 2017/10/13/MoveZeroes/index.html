Move Zeroes

2017.10.13 叶帆 LeetCode OJ  热度 ℃

题目介绍

LeetCode 283. Move Zeroes，题目的意思在不复制数组的情况下，用最少的操作将 0 移到最后。

解题思路

一个移动的 i pointer（初始值为 0），遍历数组发现非 0 元素，则将对应 i 的元素赋值为当前遍历的值。遍历完成后，比较 i 和数组长度，将剩余元素复制为 0 即可。
