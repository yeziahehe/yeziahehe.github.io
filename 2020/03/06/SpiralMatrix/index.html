Spiral Matrix

2020.03.06 叶帆 LeetCode OJ  热度 ℃

题目介绍

LeetCode 54. Spiral Matrix。

复杂度

时间复杂度: O(n), 空间复杂度: O(n)

解题思路

找到移动的条件 i+v == m.count || i+v == -1 || j+h == m[0].count || j+h == -1 || m[i+v][j+h] == 0
取出来的元素置为 0，(h, v) 组合代表移动方向

  • → (1, 0)
  • ↓ (0, 1)
  • ← (-1, 0)
  • ↑ (0, -1)
